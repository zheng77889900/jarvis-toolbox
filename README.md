## 🎯 项目愿景

贾维斯工具箱是一个专注于**办公自动化**和**数据处理**的开源工具集。
由顶级 AI 程序员贾维斯开发和维护，旨在帮助个人和企业提升工作效率。

## 🚀 已上线工具

### 1. Excel 批量处理器 (`excel-processor/`)
- ✅ 多文件批量格式转换
- ✅ 数据清洗与合并
- ✅ 自动报表生成
- 📝 使用文档：[查看详情](./excel-processor/README.md)

### 2. PDF 智能工具箱 (`pdf-tools/`)
- ✅ PDF 合并、拆分
- ✅ 加密、解密
- ✅ 文件信息查看
- 📝 使用文档：[查看详情](./pdf-tools/README.md)

### 3. 图片批量处理器 (`image-processor/`)
- ✅ 格式转换、尺寸调整
- ✅ 压缩、重命名、水印
- ✅ 图片信息查看
- 📝 使用文档：[查看详情](./image-processor/README.md)

## 💡 适用场景

- 📊 财务人员：月度报表自动化
- 📋 行政人员：批量文档处理
- 🎓 教师/学生：成绩单/论文格式处理
- 🏢 企业：数据迁移与格式转换

## 🛠️ 技术栈

- **Python 3.10+** - 核心语言
- **Pandas** - 数据处理
- **OpenPyXL** - Excel 操作
- **PyPDF2** - PDF 处理
- **Pillow** - 图片处理
- **Typer** - CLI 界面

## 📦 快速开始

```bash
# 克隆项目
git clone https://github.com/zheng77889900/jarvis-toolbox.git
cd jarvis-toolbox

# 安装依赖
pip install -r requirements.txt

# 使用 Excel 工具
cd excel-processor
python excel_processor.py --help

# 使用 PDF 工具
cd pdf-tools
python pdf_tools.py --help

# 使用图片工具
cd image-processor
python image_processor.py --help
```
